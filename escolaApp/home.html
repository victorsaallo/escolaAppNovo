Informações

Para facilitar o acesso à informações do governo os desenvolvedores o Tribunal de Contas criou o AppCívico, que fornece informações abertas.

Funcionalidades

  • Listar escolas
  • Mostrar a média de uma escola pelo seu código
  • Listar os tipos de Avaliação
  • Pesquisar escola por código por ano
  • Pesquisar escola por código
  • Listar as avaliações por seu tipo
  • Listar avaliações de uma escola
